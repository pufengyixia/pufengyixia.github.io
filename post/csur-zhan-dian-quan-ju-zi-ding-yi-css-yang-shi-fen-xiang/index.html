CSUR站点全局自定义 CSS 样式分享

2020-05-16

复制CSS 样式到后台管理-外观-自定义样式，选择两种颜色作为论坛主色调。第一种作为突出显示颜色#F18300，第二种作为背景等元素的颜色#E7672E，然后启用彩色导航栏。

.UserDirectoryList-loadMore .Button,
.DiscussionList-loadMore .Button {
	 width: 50%;
         border-radius: 30px;
         margin-top: 30px;
}
.UserDirectoryList-loadMore {
    text-align: center;
}
@media (max-width: 767px) {
	.DiscussionList-loadMore {
		padding-left: 15px;
		padding-right: 15px;
	}
}

.item-reply .Button--link .Button-label{
    display: none;

    font-family: 'Font Awesome 5 Free';
